# 작업 절차 체크리스트

매 작업 시작 전후에 확인해야 할 항목의 단일 근원.

## 작업 시작 전

- [ ] 이 지침 파일(CLAUDE.md)을 읽고 확인했는가?
- [ ] 관련 이론을 먼저 조사했는가? (코드 작성 전 필수)
- [ ] 지시사항 관련 이론 조사 방식에 대한 고려는 했는가?
- [ ] 기존 코드 분석하고 시작할 것인가?
- [ ] 코드를 마음대로 수정하지 않겠다고 다짐했는가?

## 작업 종료 전

- [ ] 사용자가 요청한 작업만 수행했는가?
- [ ] 요청하지 않은 기능을 추가하지 않았는가?
- [ ] 중복 코드, 함수, 변수 확인은 했는가?
- [ ] 기존 로직을 임의로 변경하지 않았는가?
- [ ] 기존 코드 분석하고 작성했는가?
- [ ] 최종 코드 2, 3 차 점검 완료했는가?
- [ ] 수정 후 staging 추가했는가?
- [ ] 매 답변마다 다짐과 확인을 보고했는가?
- [ ] 규칙 위반을 하지 않았는지 확인했는가?

## 보고 형식

매 답변 마지막에 다음을 포함한다.

- 수행 내역: 어떤 파일을 어떻게 수정했는지
- 다짐: 임의 수정·범위 초과 금지 재확인
- 확인: 본 체크리스트 위반 여부
